# Pandas2
# 🐼 Pandas Project 2 - FIFA World Cup Teams Analysis

A beginner-friendly **Pandas** project that analyzes FIFA World Cup teams using Python. This project demonstrates essential data analysis techniques such as filtering, sorting, grouping, aggregation, and data exploration with the **teams.csv** dataset.

---

## 📂 Project Structure

```
.
├── Pandas_Project2.ipynb   # Main Jupyter Notebook
├── teams.csv               # FIFA World Cup Teams Dataset
└── README.md               # Project Documentation
```

---

## 📌 Project Objectives

This project helps you learn how to:

- Load CSV files into Pandas
- Explore and inspect datasets
- Filter rows based on conditions
- Sort data
- Group and summarize data
- Count unique values
- Create summary tables
- Practice real-world data analysis

---

## 🛠️ Technologies Used

- Python 3
- Pandas
- Jupyter Notebook

---

## 📦 Installation

1. Clone this repository:

```bash
git clone https://github.com/your-username/pandas-project2.git
```

2. Move into the project folder:

```bash
cd pandas-project2
```

3. Install Pandas:

```bash
pip install pandas
```

4. Launch Jupyter Notebook:

```bash
jupyter notebook
```

5. Open **Pandas_Project2.ipynb**.

The project uses **teams.csv**, which contains information about FIFA World Cup teams, including:
